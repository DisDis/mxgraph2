VISA Network Editor

VISA Network Editor - Phase 1 is a read-only viewer for VISA client networks and a backend for creating graph models, and bitmaps of diagrams on the server-side.

Demos

A client-side demo with a static HTML file is available here. (Note: Most browsers will block local XHR so this demo is best viewed via a webserver.)

The server-side demo must be placed on a PHP-enabled server and is available here. GD support is required on the server to render bitmap graphics. Support for TrueType fonts is also recommended.
